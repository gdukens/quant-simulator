# Probability Theory Foundations\n\nProbability theory provides the mathematical foundation for modeling uncertainty in financial markets. This chapter establishes the rigorous probabilistic framework necessary for quantitative finance.\n\n## 1. Probability Spaces\n\n### 1.1 Basic Definitions\n\n**Definition 1.1** (Probability Space): A probability space is a triple $(\\Omega, \\mathcal{F}, \\mathbb{P})$ where:\n- $\\Omega$ is a non-empty set called the sample space\n- $\\mathcal{F}$ is a $\\sigma$-algebra of subsets of $\\Omega$ \n- $\\mathbb{P}$ is a probability measure on $(\\Omega, \\mathcal{F})$\n\n**Definition 1.2** ($\\sigma$-algebra): A collection $\\mathcal{F}$ of subsets of $\\Omega$ is a $\\sigma$-algebra if:\n1. $\\Omega \\in \\mathcal{F}$\n2. If $A \\in \\mathcal{F}$, then $A^c \\in \\mathcal{F}$\n3. If $(A_n)_{n \\geq 1} \\subset \\mathcal{F}$, then $\\bigcup_{n=1}^{\\infty} A_n \\in \\mathcal{F}$\n\n**Definition 1.3** (Probability Measure): A function $\\mathbb{P}: \\mathcal{F} \\to [0,1]$ is a probability measure if:\n1. $\\mathbb{P}(\\Omega) = 1$\n2. For any sequence $(A_n)_{n \\geq 1}$ of pairwise disjoint sets in $\\mathcal{F}$:\n\n```{math}\n:label: countable_additivity\n\n\\mathbb{P}\\left(\\bigcup_{n=1}^{\\infty} A_n\\right) = \\sum_{n=1}^{\\infty} \\mathbb{P}(A_n)\n```\n\n### 1.2 Financial Interpretation\n\nIn financial modeling:\n- $\\Omega$ represents all possible market scenarios over a given time horizon\n- Events in $\\mathcal{F}$ represent observable market conditions\n- $\\mathbb{P}$ assigns probabilities to these market conditions\n\n**Example 1.1**: Consider a two-period binomial model for a stock price:\n- $\\Omega = \\{uu, ud, du, dd\\}$ (up-up, up-down, down-up, down-down)\n- $\\mathcal{F} = 2^\\Omega$ (all subsets)\n- Under risk-neutral measure: $\\mathbb{P}(\\{uu\\}) = q^2$, $\\mathbb{P}(\\{ud\\}) = q(1-q)$, etc.\n\nwhere $q$ is the risk-neutral probability of an up move.\n\n## 2. Random Variables and Distributions\n\n### 2.1 Random Variables\n\n**Definition 2.1** (Random Variable): A random variable is a measurable function $X: \\Omega \\to \\mathbb{R}$:\n\n```{math}\n:label: measurable_function\n\n\\{\\omega \\in \\Omega : X(\\omega) \\leq x\\} \\in \\mathcal{F} \\quad \\forall x \\in \\mathbb{R}\n```\n\n**Definition 2.2** (Distribution Function): The cumulative distribution function (CDF) of $X$ is:\n\n```{math}\n:label: cdf\n\nF_X(x) = \\mathbb{P}(X \\leq x) = \\mathbb{P}(\\{\\omega : X(\\omega) \\leq x\\})\n```\n\n### 2.2 Moments and Characteristic Functions\n\n**Definition 2.3** (Moments): For $k \\geq 1$, the $k$-th moment of $X$ is:\n\n```{math}\n:label: moments\n\nm_k = \\mathbb{E}[X^k] = \\int_{-\\infty}^{\\infty} x^k dF_X(x)\n```\n\nprovided the integral exists.\n\n**Key Financial Moments**:\n- First moment: $\\mu = \\mathbb{E}[X]$ (expected return)\n- Second central moment: $\\sigma^2 = \\text{Var}[X] = \\mathbb{E}[(X - \\mu)^2]$ (volatility squared)\n- Third central moment: $\\mathbb{E}[(X - \\mu)^3]$ (related to skewness)\n- Fourth central moment: $\\mathbb{E}[(X - \\mu)^4]$ (related to kurtosis)\n\n**Definition 2.4** (Characteristic Function): The characteristic function of $X$ is:\n\n```{math}\n:label: characteristic_function\n\n\\phi_X(t) = \\mathbb{E}[e^{itX}] = \\int_{-\\infty}^{\\infty} e^{itx} dF_X(x)\n```\n\n### 2.3 Important Distributions in Finance\n\n#### 2.3.1 Normal Distribution\n\nThe normal distribution $X \\sim \\mathcal{N}(\\mu, \\sigma^2)$ has density:\n\n```{math}\n:label: normal_density\n\nf_X(x) = \\frac{1}{\\sqrt{2\\pi\\sigma^2}} \\exp\\left(-\\frac{(x-\\mu)^2}{2\\sigma^2}\\right)\n```\n\n**Properties**:\n- Characteristic function: $\\phi_X(t) = \\exp(i\\mu t - \\frac{1}{2}\\sigma^2 t^2)$\n- Moment generating function: $M_X(t) = \\exp(\\mu t + \\frac{1}{2}\\sigma^2 t^2)$\n\n#### 2.3.2 Log-Normal Distribution\n\nIf $\\log X \\sim \\mathcal{N}(\\mu, \\sigma^2)$, then $X$ has log-normal distribution with density:\n\n```{math}\n:label: lognormal_density\n\nf_X(x) = \\frac{1}{x\\sqrt{2\\pi\\sigma^2}} \\exp\\left(-\\frac{(\\log x - \\mu)^2}{2\\sigma^2}\\right), \\quad x > 0\n```\n\n**Financial Application**: Under geometric Brownian motion, asset prices follow log-normal distributions.\n\n#### 2.3.3 Student's t-Distribution\n\nThe $t$-distribution with $\\nu$ degrees of freedom has density:\n\n```{math}\n:label: t_density\n\nf_X(x) = \\frac{\\Gamma(\\frac{\\nu+1}{2})}{\\sqrt{\\nu\\pi}\\Gamma(\\frac{\\nu}{2})}\\left(1 + \\frac{x^2}{\\nu}\\right)^{-\\frac{\\nu+1}{2}}\n```\n\n**Financial Application**: Heavy-tailed distributions for modeling extreme market movements.\n\n## 3. Conditional Expectation and Information\n\n### 3.1 Conditional Expectation\n\n**Definition 3.1** (Conditional Expectation): Given a sub-$\\sigma$-algebra $\\mathcal{G} \\subseteq \\mathcal{F}$ and an integrable random variable $X$, the conditional expectation $\\mathbb{E}[X|\\mathcal{G}]$ is the unique (up to sets of measure zero) $\\mathcal{G}$-measurable random variable such that:\n\n```{math}\n:label: conditional_expectation\n\n\\int_G \\mathbb{E}[X|\\mathcal{G}] d\\mathbb{P} = \\int_G X d\\mathbb{P} \\quad \\forall G \\in \\mathcal{G}\n```\n\n**Properties**:\n1. **Linearity**: $\\mathbb{E}[aX + bY|\\mathcal{G}] = a\\mathbb{E}[X|\\mathcal{G}] + b\\mathbb{E}[Y|\\mathcal{G}]$\n2. **Tower Property**: $\\mathbb{E}[\\mathbb{E}[X|\\mathcal{G}]] = \\mathbb{E}[X]$\n3. **Taking Out Known**: If $Y$ is $\\mathcal{G}$-measurable, then $\\mathbb{E}[XY|\\mathcal{G}] = Y\\mathbb{E}[X|\\mathcal{G}]$\n\n### 3.2 Filtrations and Adapted Processes\n\n**Definition 3.2** (Filtration): A filtration is a family $\\mathbb{F} = (\\mathcal{F}_t)_{t \\geq 0}$ of sub-$\\sigma$-algebras of $\\mathcal{F}$ such that:\n\n```{math}\n:label: filtration_property\n\n\\mathcal{F}_s \\subseteq \\mathcal{F}_t \\subseteq \\mathcal{F} \\quad \\text{whenever } 0 \\leq s \\leq t\n```\n\n**Financial Interpretation**: $\\mathcal{F}_t$ represents all information available at time $t$.\n\n**Definition 3.3** (Adapted Process): A stochastic process $(X_t)_{t \\geq 0}$ is adapted to the filtration $\\mathbb{F}$ if $X_t$ is $\\mathcal{F}_t$-measurable for all $t \\geq 0$.\n\n## 4. Convergence of Random Variables\n\n### 4.1 Types of Convergence\n\n**Definition 4.1** (Almost Sure Convergence): $(X_n)$ converges to $X$ almost surely if:\n\n```{math}\n:label: as_convergence\n\n\\mathbb{P}(\\{\\omega : \\lim_{n \\to \\infty} X_n(\\omega) = X(\\omega)\\}) = 1\n```\n\n**Definition 4.2** (Convergence in Probability): $(X_n)$ converges to $X$ in probability if:\n\n```{math}\n:label: prob_convergence\n\n\\lim_{n \\to \\infty} \\mathbb{P}(|X_n - X| > \\epsilon) = 0 \\quad \\forall \\epsilon > 0\n```\n\n**Definition 4.3** (Convergence in Distribution): $(X_n)$ converges to $X$ in distribution if:\n\n```{math}\n:label: dist_convergence\n\n\\lim_{n \\to \\infty} F_{X_n}(x) = F_X(x)\n```\n\nat all continuity points of $F_X$.\n\n### 4.2 Limit Theorems\n\n**Theorem 4.1** (Strong Law of Large Numbers): If $(X_n)$ are i.i.d. with $\\mathbb{E}[|X_1|] < \\infty$, then:\n\n```{math}\n:label: slln\n\n\\frac{1}{n}\\sum_{i=1}^n X_i \\to \\mathbb{E}[X_1] \\quad \\text{a.s.}\n```\n\n**Theorem 4.2** (Central Limit Theorem): If $(X_n)$ are i.i.d. with $\\mathbb{E}[X_1] = \\mu$ and $\\text{Var}[X_1] = \\sigma^2 < \\infty$, then:\n\n```{math}\n:label: clt\n\n\\sqrt{n}\\left(\\frac{1}{n}\\sum_{i=1}^n X_i - \\mu\\right) \\xrightarrow{d} \\mathcal{N}(0, \\sigma^2)\n```\n\n## 5.
