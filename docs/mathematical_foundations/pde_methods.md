# Partial Differential Equations in Finance\n\nPartial differential equations (PDEs) provide an alternative to Monte Carlo methods for pricing derivatives and analyzing financial instruments. This chapter presents the mathematical theory of PDEs in finance and their numerical solution techniques.\n\n## 1. The Black-Scholes PDE\n\n### 1.1 Derivation from No-Arbitrage Principle\n\n**Setup**: Consider a European option with payoff $H(S_T)$ on an asset following:\n\n```{math}\n:label: asset_dynamics\n\ndS_t = \\mu S_t dt + \\sigma S_t dW_t\n```\n\n**Risk-Free Portfolio Construction**: Form a portfolio $\\Pi_t$ with:\n- Long one option: $V(t, S_t)$\n- Short $\\Delta$ units of stock: $-\\Delta S_t$\n\n**Portfolio Value**: $\\Pi_t = V(t, S_t) - \\Delta S_t$\n\n**Portfolio Dynamics** (using Itô's formula):\n\n```{math}\n:label: portfolio_dynamics\n\nd\\Pi_t = \\left(\\frac{\\partial V}{\\partial t} + \\frac{1}{2}\\sigma^2 S_t^2 \\frac{\\partial^2 V}{\\partial S^2}\\right) dt + \\left(\\frac{\\partial V}{\\partial S} - \\Delta\\right) dS_t\n```\n\n**Delta Hedging**: Choose $\\Delta = \\frac{\\partial V}{\\partial S}$ to eliminate stochastic term:\n\n```{math}\n:label: delta_hedge\n\nd\\Pi_t = \\left(\\frac{\\partial V}{\\partial t} + \\frac{1}{2}\\sigma^2 S_t^2 \\frac{\\partial^2 V}{\\partial S^2}\\right) dt\n```\n\n**No-Arbitrage Condition**: The risk-free portfolio must earn the risk-free rate:\n\n```{math}\n:label: no_arbitrage_portfolio\n\nd\\Pi_t = r \\Pi_t dt = r\\left(V - S_t \\frac{\\partial V}{\\partial S}\\right) dt\n```\n\n**Black-Scholes PDE**:\n\n```{math}\n:label: black_scholes_pde\n\n\\frac{\\partial V}{\\partial t} + \\frac{1}{2}\\sigma^2 S^2 \\frac{\\partial^2 V}{\\partial S^2} + r S \\frac{\\partial V}{\\partial S} - r V = 0\n```\n\nwith terminal condition $V(T, S) = H(S)$.\n\n### 1.2 Connection to Feynman-Kac Formula\n\nThe PDE solution is equivalent to the risk-neutral expectation:\n\n```{math}\n:label: feynman_kac_bs\n\nV(t, S) = \\mathbb{E}^\\mathbb{Q}\\left[e^{-r(T-t)} H(S_T) \\big| S_t = S\\right]\n```\n\nwhere under $\\mathbb{Q}$: $dS_t = rS_t dt + \\sigma S_t dW_t^\\mathbb{Q}$.\n\n### 1.3 Boundary Conditions\n\n**European Call Option** ($H(S) = \\max(S-K, 0)$):\n- $V(t, 0) = 0$ (worthless if asset price is zero)\n- $V(t, S) \\sim S - Ke^{-r(T-t)}$ as $S \\to \\infty$ (intrinsic value)\n- $V(T, S) = \\max(S-K, 0)$ (terminal condition)\n\n**European Put Option** ($H(S) = \\max(K-S, 0)$):\n- $V(t, 0) = Ke^{-r(T-t)}$ (discounted strike)\n- $V(t, S) \\to 0$ as $S \\to \\infty$ (worthless for large asset prices)\n- $V(T, S) = \\max(K-S, 0)$ (terminal condition)\n\n## 2. Generalized PDE Framework\n\n### 2.1 Multi-Asset Black-Scholes PDE\n\nFor $n$ correlated assets with dynamics:\n\n```{math}\n:label: multi_asset_dynamics\n\ndS_t^i = r S_t^i dt + \\sum_{j=1}^n \\sigma_{ij} S_t^i dW_t^j\n```\n\nThe PDE becomes:\n\n```{math}\n:label: multi_asset_pde\n\n\\frac{\\partial V}{\\partial t} + \\frac{1}{2} \\sum_{i,j=1}^n \\rho_{ij} \\sigma_i \\sigma_j S_i S_j \\frac{\\partial^2 V}{\\partial S_i \\partial S_j} + r \\sum_{i=1}^n S_i \\frac{\\partial V}{\\partial S_i} - r V = 0\n```\n\nwhere $\\rho_{ij}$ is the correlation between assets $i$ and $j$.\n\n### 2.2 Stochastic Volatility Models\n\n**Heston Model**: For $dS_t = rS_t dt + \\sqrt{v_t} S_t dW_t^S$ and $dv_t = \\kappa(\\theta - v_t) dt + \\xi \\sqrt{v_t} dW_t^v$ with $dW_t^S dW_t^v = \\rho dt$:\n\n```{math}\n:label: heston_pde\n\n\\frac{\\partial V}{\\partial t} + \\frac{1}{2} v S^2 \\frac{\\partial^2 V}{\\partial S^2} + \\rho \\xi v S \\frac{\\partial^2 V}{\\partial S \\partial v} + \\frac{1}{2} \\xi^2 v \\frac{\\partial^2 V}{\\partial v^2} + r S \\frac{\\partial V}{\\partial S} + \\kappa(\\theta - v) \\frac{\\partial V}{\\partial v} - r V = 0\n```\n\n### 2.3 Jump-Diffusion Models\n\n**Merton Jump-Diffusion**: With Poisson jumps at rate $\\lambda$ and log-normal jump sizes:\n\n```{math}\n:label: jump_diffusion_pde\n\n\\frac{\\partial V}{\\partial t} + \\frac{1}{2}\\sigma^2 S^2 \\frac{\\partial^2 V}{\\partial S^2} + r S \\frac{\\partial V}{\\partial S} - r V + \\lambda \\int_{-\\infty}^{\\infty} [V(t, Se^y) - V(t,S)] f(y) dy = 0\n```\n\nwhere $f(y)$ is the jump size distribution density.\n\n## 3. American Options and Free Boundary Problems\n\n### 3.1 American Option Formulation\n\n**Optimal Stopping Problem**: American options involve finding the optimal exercise policy.\n\n**Free Boundary**: The exercise boundary $S_f(t)$ separates continuation and exercise regions.\n\n**Linear Complementarity Problem**: For an American put:\n\n```{math}\n:label: american_lcp\n\n\\begin{align}\n\\mathcal{L}V &\\geq 0 \\quad \\text{(PDE satisfied)} \\\\\nV &\\geq \\max(K-S, 0) \\quad \\text{(value constraint)} \\\\\n(\\mathcal{L}V)(V - \\max(K-S, 0)) &= 0 \\quad \\text{(complementarity)}\nend{align}\n```\n\nwhere $\\mathcal{L}$ is the Black-Scholes operator.\n\n### 3.2 Free Boundary Conditions\n\nAt the exercise boundary $S = S_f(t)$:\n\n1. **Value Matching**: $V(t, S_f(t)) = K - S_f(t)$\n2. **Smooth Pasting**: $\\frac{\\partial V}{\\partial S}(t, S_f(t)) = -1$\n\n**High Contact Condition**: For perpetual American options:\n\n```{math}\n:label: high_contact\n\n\\frac{\\partial^2 V}{\\partial S^2}(S_f) = 0\n```\n\n## 4. Numerical Methods for PDEs\n\n### 4.1 Finite Difference Methods\n\n#### 4.1.1 Grid Construction\n\n**Spatial Grid**: $S_{min} = 0 < S_1 < S_2 < \\cdots < S_M = S_{max}$\n\n**Time Grid**: $0 = t_0 < t_1 < \\cdots < t_N = T$\n\n**Notation**: $V_i^n \\approx V(t_n, S_i)$\n\n#### 4.1.2 Finite Difference Approximations\n\n**First Derivatives**:\n- Forward: $\\frac{\\partial V}{\\partial S} \\approx \\frac{V_{i+1}^n - V_i^n}{\\Delta S}$\n- Backward: $\\frac{\\partial V}{\\partial S} \\approx \\frac{V_i^n - V_{i-1}^n}{\\Delta S}$\n- Central: $\\frac{\\partial V}{\\partial S} \\approx \\frac{V_{i+1}^n - V_{i-1}^n}{2\\Delta S}$\n\n**Second Derivative**:\n\n```{math}\n:label: second_derivative_fd\n\n\\frac{\\partial^2 V}{\\partial S^2} \\approx \\frac{V_{i+1}^n - 2V_i^n + V_{i-1}^n}{(\\Delta S)^2}\n```\n\n**Time Derivative**:\n\n```{math}\n:label: time_derivative_fd\n\n\\frac{\\partial V}{\\partial t} \\approx \\frac{V_i^{n+1} - V_i^n}{\\Delta t}\n```\n\n#### 4.1.3 Explicit Finite Difference Scheme\n\nSubstitute approximations into Black-Scholes PDE:\n\n```{math}\n:label: explicit_scheme\n\n\\frac{V_i^{n+1} - V_i^n}{\\Delta t} + \\frac{1}{2}\\sigma^2 S_i^2 \\frac{V_{i+1}^n - 2V_i^n + V_{i-1}^n}{(\\Delta S)^2} + r S_i \\frac{V_{i+1}^n - V_{i-1}^n}{2\\Delta S} - r V_i^n = 0\n```\n\n**Stability Condition**: For stability, require:\n\n```{math}\n:label: stability_condition\n\n\\Delta t \\leq \\frac{(\\Delta S)^2}{\\sigma^2 S_{max}^2 + r|S_{max}|\\Delta S}\n```\n\n#### 4.1.4 Implicit Finite Difference Scheme\n\nEvaluate spatial derivatives at time $n+1$:\n\n```{math}\n:label: implicit_scheme\n\n\\frac{V_i^{n+1} - V_i^n}{\\Delta t} + \\frac{1}{2}\\sigma^2 S_i^2 \\frac{V_{i+1}^{n+1} - 2V_i^{n+1} + V_{i-1}^{n+1}}{(\\Delta S)^2} + r S_i \\frac{V_{i+1}^{n+1} - V_{i-1}^{n+1}}{2\\Delta S} - r V_i^{n+1} = 0\n```\n\n**Matrix Form**: $AV^{n+1} = V^n$ where $A$ is tridiagonal.\n\n**Advantages**: Unconditionally stable, larger time steps possible.\n\n#### 4.1.5 Crank-Nicolson Scheme\n\nAverage of explicit and implicit schemes:\n\n```{math}\n:label: crank_nicolson\n\n\\frac{V_i^{n+1} - V_i^n}{\\Delta t} = \\frac{1}{2}\\left[\\mathcal{L}V_i^{n+1} + \\mathcal{L}V_i^n\\right]\n```\n\nwhere $\\mathcal{L}$ is the spatial differential operator.\n\n**Properties**: Second-order accurate in time, unconditionally stable.\n\n### 4.2 Finite Element Methods\n\n#### 4.2.1 Weak Formulation\n\nMultiply PDE by test function $\\phi$ and integrate by parts:\n\n```{math}\n:label: weak_formulation\n\n\\int_\\Omega \\frac{\\partial V}{\\partial t} \\phi \\, dS + \\int_\\Omega \\frac{1}{2}\\sigma^2 S^2 \\frac{\\partial V}{\\partial S} \\frac{\\partial \\phi}{\\partial S} \\, dS + \\int_\\Omega \\left(rS \\frac{\\partial V}{\\partial S} - rV\\right) \\phi \\, dS = 0\n```\n\n#### 4.2.2 Basis Functions\n\n**Linear Elements**: On interval $[S_i, S_{i+1}]$:\n\n```{math}\n:label: linear_basis\n\n\\phi_i(S) = \\begin{cases}\n\\frac{S - S_{i-1}}{S_i - S_{i-1}} & S \\in [S_{i-1}, S_i] \\\\\n\\frac{S_{i+1} - S}{S_{i+1} - S_i} & S \\in [S_i, S_{i+1}] \\\\\n0 & \\text{otherwise}\nend{cases}\n```\n\n#### 4.2.3 Assembly and Solution\n\nLead to system: $M\\frac{dV}{dt} + KV = F$ where:\n- $M$: Mass matrix\n- $K$: Stiffness matrix  \n- $F$: Load vector\n\n### 4.3 Spectral Methods\n\n#### 4.3.1 Fourier Transform Methods\n\nFor infinite domains, use Fourier transforms to convert PDE to ODE.\n\n**Example**: For constant coefficients, the fundamental solution involves:\n\n```{math}\n:label: fundamental_solution\n\nG(S, t; S_0, 0) = \\frac{1}{\\sqrt{2\\pi \\sigma^2 t}} \\exp\\left(-\\frac{(\\ln(S/S_0) - (r-\\sigma^2/2)t)^2}{2\\sigma^2 t}\\right)\n```\n\n#### 4.3.2 Chebyshev Polynomial Methods\n\nApproximate solution using Chebyshev polynomials:\n\n```{math}\n:label: chebyshev_approximation\n\nV(t, S) \\approx \\sum_{k=0}^N a_k(t) T_k\\left(\\frac{2S - (S_{max} + S_{min})}{S_{max} - S_{min}}\\right)\n```\n\nwhere $T_k$ are Chebyshev polynomials.\n\n## 5. Advanced PDE Techniques\n\n### 5.1 Coordinate Transformations\n\n#### 5.1.1 Log-Price Transformation\n\nLet $x = \\ln S$ and $u(t, x) = V(t, e^x)$. The Black-Scholes PDE becomes:\n\n```{math}\n:label: log_price_pde\n\n\\frac{\\partial u}{\\partial t} + \\frac{1}{2}\\sigma^2 \\frac{\\partial^2 u}{\\partial x^2} + \\left(r - \\frac{\\sigma^2}{2}\\right) \\frac{\\partial u}{\\partial x} - r u = 0\n```\n\n**Advantages**: Constant coefficients, simpler numerical treatment.\n\n#### 5.1.2 Similarity Reduction\n\nFor perpetual options, introduce similarity variable $\\eta = S/K$ to reduce dimensionality.\n\n### 5.2 Asymptotic Methods\n\n#### 5.2.1 Small Volatility Expansion\n\nFor small $\\sigma$, expand: $V = V_0 + \\sigma V_1 + \\sigma^2 V_2 + \\cdots$\n\n**Leading Order**: $V_0 = e^{-r(T-t)} \\max(S - K, 0)$ (intrinsic value)\n\n#### 5.2.2 Large Time Expansion\n\nFor large maturity $T$, different asymptotic behavior emerges.\n\n### 5.3 Moving Boundary Methods\n\n#### 5.3.1 Front-Tracking Methods\n\nExplicitly track the free boundary location using additional equations.\n\n#### 5.3.2 Penalty Methods\n\nApproximate the complementarity condition with penalty terms:\n\n```{math}\n:label: penalty_method\n\n\\mathcal{L}V - r V + \\frac{1}{\\epsilon} \\max(0, \\max(K-S, 0) - V) = 0\n```\n\nwhere $\\epsilon \\to 0$.\n\n## 6.
