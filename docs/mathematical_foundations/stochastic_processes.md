# Stochastic Processes in Finance\n\nStochastic processes provide the mathematical framework for modeling asset price evolution over time. This chapter develops the theory of stochastic processes with particular emphasis on their applications in quantitative finance.\n\n## 1. Basic Definitions and Properties\n\n### 1.1 Stochastic Processes\n\n**Definition 1.1** (Stochastic Process): A stochastic process is a collection $(X_t)_{t \\in T}$ of random variables indexed by a parameter $t$ (usually time), all defined on the same probability space $(\\Omega, \\mathcal{F}, \\mathbb{P})$.\n\n**Definition 1.2** (Sample Path): For each $\\omega \\in \\Omega$, the function $t \\mapsto X_t(\\omega)$ is called a sample path or trajectory of the process.\n\n### 1.2 Classification of Stochastic Processes\n\n#### 1.2.1 By State Space\n- **Discrete state space**: $X_t \\in \\{0, 1, 2, \\ldots\\}$ (e.g., number of trades)\n- **Continuous state space**: $X_t \\in \\mathbb{R}$ (e.g., asset prices)\n\n#### 1.2.2 By Time Parameter\n- **Discrete time**: $t \\in \\{0, 1, 2, \\ldots\\}$ (e.g., daily returns)\n- **Continuous time**: $t \\in [0, \\infty)$ (e.g., high-frequency price processes)\n\n#### 1.2.3 By Dependence Structure\n- **Independent increments**: Future changes independent of past given current state\n- **Markov property**: Future depends only on current state, not history\n- **Martingale property**: Expected future value equals current value\n\n## 2. Brownian Motion\n\n### 2.1 Definition and Construction\n\n**Definition 2.1** (Standard Brownian Motion): A stochastic process $(W_t)_{t \\geq 0}$ is a standard Brownian motion if:\n1. $W_0 = 0$ (starts at zero)\n2. $(W_t)_{t \\geq 0}$ has independent increments\n3. For $0 \\leq s < t$: $W_t - W_s \\sim \\mathcal{N}(0, t-s)$\n4. Sample paths are continuous\n\n**Properties**:\n- **Gaussian process**: All finite-dimensional distributions are multivariate normal\n- **Covariance function**: $\\text{Cov}[W_s, W_t] = \\min(s,t)$\n- **Quadratic variation**: $[W]_t = t$ (deterministic)\n- **Nowhere differentiable**: Sample paths are continuous but nowhere differentiable\n\n### 2.2 Brownian Motion with Drift\n\n**Definition 2.2** (Brownian Motion with Drift): The process:\n\n```{math}\n:label: bm_drift\n\nX_t = \\mu t + \\sigma W_t\n```\n\nwhere $\\mu \\in \\mathbb{R}$ (drift) and $\\sigma > 0$ (volatility).\n\n**Properties**:\n- $\\mathbb{E}[X_t] = \\mu t$\n- $\\text{Var}[X_t] = \\sigma^2 t$\n- Increments: $X_t - X_s \\sim \\mathcal{N}(\\mu(t-s), \\sigma^2(t-s))$\n\n### 2.3 Geometric Brownian Motion\n\n**Definition 2.3** (Geometric Brownian Motion): The process:\n\n```{math}\n:label: gbm\n\nS_t = S_0 \\exp\\left(\\left(\\mu - \\frac{\\sigma^2}{2}\\right)t + \\sigma W_t\\right)\n```\n\n**Differential Form**:\n\n```{math}\n:label: gbm_sde\n\ndS_t = \\mu S_t dt + \\sigma S_t dW_t\n```\n\n**Properties**:\n- $S_t > 0$ for all $t$ (non-negative prices)\n- Log-returns are normally distributed: $\\log(S_t/S_s) \\sim \\mathcal{N}((\\mu - \\sigma^2/2)(t-s), \\sigma^2(t-s))$\n- $\\mathbb{E}[S_t] = S_0 e^{\\mu t}$\n- $\\text{Var}[S_t] = S_0^2 e^{2\\mu t}(e^{\\sigma^2 t} - 1)$\n\n## 3. Markov Processes\n\n### 3.1 Markov Property\n\n**Definition 3.1** (Markov Process): A process $(X_t)_{t \\geq 0}$ satisfies the Markov property if:\n\n```{math}\n:label: markov_property\n\n\\mathbb{P}(X_t \\in A | \\mathcal{F}_s) = \\mathbb{P}(X_t \\in A | X_s) \\quad \\text{for all } 0 \\leq s \\leq t\n```\n\n**Intuition**: Given the present state, the future is independent of the past.\n\n### 3.2 Transition Probabilities\n\n**Definition 3.2** (Transition Function): For a Markov process:\n\n```{math}\n:label: transition_function\n\nP_t(x, A) = \\mathbb{P}(X_{s+t} \\in A | X_s = x)\n```\n\n**Chapman-Kolmogorov Equation**:\n\n```{math}\n:label: chapman_kolmogorov\n\nP_{s+t}(x, A) = \\int P_t(y, A) P_s(x, dy)\n```\n\n### 3.3 Infinitesimal Generator\n\n**Definition 3.3** (Infinitesimal Generator): For a Markov process with transition semigroup $(P_t)_{t \\geq 0}$:\n\n```{math}\n:label: generator\n\n\\mathcal{L}f(x) = \\lim_{t \\downarrow 0} \\frac{P_t f(x) - f(x)}{t}\n```\n\nwhere $(P_t f)(x) = \\int f(y) P_t(x, dy)$.\n\n**For Diffusion Processes**: If $dX_t = \\mu(X_t) dt + \\sigma(X_t) dW_t$, then:\n\n```{math}\n:label: diffusion_generator\n\n\\mathcal{L}f(x) = \\mu(x) f'(x) + \\frac{1}{2}\\sigma^2(x) f''(x)\n```\n\n## 4. Martingales\n\n### 4.1 Martingale Definition\n\n**Definition 4.1** (Martingale): A process $(M_t)_{t \\geq 0}$ adapted to filtration $(\\mathcal{F}_t)_{t \\geq 0}$ is a martingale if:\n1. $\\mathbb{E}[|M_t|] < \\infty$ for all $t \\geq 0$\n2. $\\mathbb{E}[M_t | \\mathcal{F}_s] = M_s$ for all $0 \\leq s \\leq t$\n\n**Sub/Supermartingales**: Replace equality with $\\geq$ (submartingale) or $\\leq$ (supermartingale).\n\n### 4.2 Examples of Martingales\n\n#### 4.2.1 Brownian Motion\n$W_t$ is a martingale with respect to its natural filtration.\n\n#### 4.2.2 Exponential Martingale\nFor any $\\theta \\in \\mathbb{R}$:\n\n```{math}\n:label: exp_martingale\n\nM_t = \\exp\\left(\\theta W_t - \\frac{\\theta^2}{2} t\\right)\n```\n\nis a martingale.\n\n#### 4.2.3 Discounted Asset Prices\nUnder the risk-neutral measure, discounted asset prices are martingales:\n\n```{math}\n:label: discounted_martingale\n\ne^{-rt} S_t \\text{ is a } \\mathbb{Q}\\text{-martingale}\n```\n\n### 4.3 Stopping Times and Optional Stopping\n\n**Definition 4.3** (Stopping Time): A random variable $\\tau: \\Omega \\to [0, \\infty]$ is a stopping time if:\n\n```{math}\n:label: stopping_time\n\n\\{\\tau \\leq t\\} \\in \\mathcal{F}_t \\quad \\forall t \\geq 0\n```\n\n**Doob's Optional Stopping Theorem**: If $(M_t)$ is a martingale and $\\tau$ is a bounded stopping time, then:\n\n```{math}\n:label: optional_stopping\n\n\\mathbb{E}[M_\\tau] = \\mathbb{E}[M_0] = M_0\n```\n\n## 5. Stochastic Differential Equations\n\n### 5.1 Formal Definition\n\n**Stochastic Differential Equation (SDE)**:\n\n```{math}\n:label: sde\n\ndX_t = \\mu(t, X_t) dt + \\sigma(t, X_t) dW_t\n```\n\nwith initial condition $X_0 = x_0$.\n\n**Integral Form**:\n\n```{math}\n:label: sde_integral\n\nX_t = X_0 + \\int_0^t \\mu(s, X_s) ds + \\int_0^t \\sigma(s, X_s) dW_s\n```\n\n### 5.2 Existence and Uniqueness\n\n**Theorem 5.1** (Existence and Uniqueness): Under Lipschitz and growth conditions:\n1. **Lipschitz**: $|\\mu(t,x) - \\mu(t,y)| + |\\sigma(t,x) - \\sigma(t,y)| \\leq L|x-y|$\n2. **Growth**: $|\\mu(t,x)|^2 + |\\sigma(t,x)|^2 \\leq K(1 + |x|^2)$\n\nthe SDE has a unique strong solution.\n\n### 5.3 Common Financial SDEs\n\n#### 5.3.1 Ornstein-Uhlenbeck Process (Mean Reversion)\n\n```{math}\n:label: ou_process\n\ndX_t = -\\lambda(X_t - \\theta) dt + \\sigma dW_t\n```\n\n**Solution**:\n\n```{math}\n:label: ou_solution\n\nX_t = \\theta + (X_0 - \\theta)e^{-\\lambda t} + \\sigma \\int_0^t e^{-\\lambda(t-s)} dW_s\n```\n\n**Applications**: Interest rate models, volatility models.\n\n#### 5.3.2 Cox-Ingersoll-Ross (CIR) Model\n\n```{math}\n:label: cir_model\n\ndr_t = \\alpha(\\beta - r_t) dt + \\sigma \\sqrt{r_t} dW_t\n```\n\n**Properties**:\n- Mean reversion to $\\beta$\n- Square-root diffusion ensures $r_t \\geq 0$\n- Used for interest rate modeling\n\n#### 5.3.3 Heston Stochastic Volatility Model\n\n```{math}\n:label: heston_model\n\n\\begin{align}\ndS_t &= rS_t dt + \\sqrt{v_t} S_t dW_t^S \\\\\ndv_t &= \\kappa(\\theta - v_t) dt + \\xi \\sqrt{v_t} dW_t^v\nend{align}\n```\n\nwhere $dW_t^S dW_t^v = \\rho dt$.\n\n## 6. Jump Processes\n\n### 6.1 Poisson Process\n\n**Definition 6.1** (Poisson Process): A counting process $(N_t)_{t \\geq 0}$ is a Poisson process with rate $\\lambda > 0$ if:\n1. $N_0 = 0$\n2. Independent increments\n3. $N_t - N_s \\sim \\text{Poisson}(\\lambda(t-s))$ for $s < t$\n\n**Properties**:\n- $\\mathbb{E}[N_t] = \\text{Var}[N_t] = \\lambda t$\n- Inter-arrival times are exponentially distributed with rate $\\lambda$\n\n### 6.2 Compound Poisson Process\n\n**Definition 6.2**: A compound Poisson process is:\n\n```{math}\n:label: compound_poisson\n\nX_t = \\sum_{i=1}^{N_t} Y_i\n```\n\nwhere $(N_t)$ is Poisson with rate $\\lambda$ and $(Y_i)$ are i.i.d. jump sizes.\n\n### 6.3 Lévy Processes\n\n**Definition 6.3** (Lévy Process): A process $(X_t)_{t \\geq 0}$ is a Lévy process if:\n1. $X_0 = 0$\n2. Independent increments\n3. Stationary increments\n4. Stochastically continuous\n\n**Lévy-Khintchine Representation**: The characteristic function is:\n\n```{math}\n:label: levy_khintchine\n\n\\mathbb{E}[e^{izX_t}] = \\exp\\left(t\\left(i\\gamma z - \\frac{\\sigma^2 z^2}{2} + \\int_{\\mathbb{R}}(e^{izx} - 1 - izx\\mathbf{1}_{|x|<1})\\nu(dx)\\right)\\right)\n```\n\nwhere $\\nu$ is the Lévy measure.\n\n## 7. Numerical Methods for Stochastic Processes\n\n### 7.1 Euler-Maruyama Scheme\n\nFor SDE $dX_t = \\mu(X_t) dt + \\sigma(X_t) dW_t$:\n\n```{math}\n:label: euler_maruyama\n\nX_{n+1} = X_n + \\mu(X_n) \\Delta t + \\sigma(X_n) \\sqrt{\\Delta t} Z_n\n```\n\nwhere $Z_n \\sim \\mathcal{N}(0,1)$ are independent.\n\n### 7.2 Milstein Scheme\n\nHigher-order accuracy:\n\n```{math}\n:label: milstein\n\nX_{n+1} = X_n + \\mu(X_n) \\Delta t + \\sigma(X_n) \\sqrt{\\Delta t} Z_n + \\frac{1}{2}\\sigma(X_n)\\sigma'(X_n)(Z_n^2 - 1)\\Delta t\n```\n\n## 8. Implementation in QuantLib Pro\n\n### 8.1 Stochastic Process Simulation\n\n```python\nfrom quantlib_pro.stochastic import StochasticProcessSimulator\nimport numpy as np\n\n# Geometric Brownian Motion\nclass GeometricBrownianMotion:\n    def __init__(self, mu, sigma, S0):\n        self.mu = mu\n        self.sigma = sigma\n        self.S0 = S0\n    \n    def simulate(self, T, n_steps, n_paths):\n        dt = T / n_steps\n        dW = np.random.normal(0, np.sqrt(dt), (n_paths, n_steps))\n        \n        # Log price process\n        log_S = np.zeros((n_paths, n_steps + 1))\n        log_S[:, 0] = np.log(self.S0)\n        \n        for i in range(n_steps):\n            log_S[:, i+1] = (log_S[:, i] + \n                           (self.mu - 0.5 * self.sigma**2) * dt + \n                           self.sigma * dW[:, i])\n        \n        return np.exp(log_S)\n\n# Usage\ngbm = GeometricBrownianMotion(mu=0.05, sigma=0.2, S0=100)\npaths = gbm.simulate(T=1.0, n_steps=252, n_paths=1000)\n```\n\n### 8.2 SDE Solver Framework\n\n```python\nfrom quantlib_pro.sde import SDESolver\n\nclass HestonModel(SDESolver):\n    def __init__(self, kappa, theta, xi, rho, r):\n        self.kappa = kappa  # Mean reversion speed\n        self.theta = theta  # Long-term volatility\n        self.xi = xi       # Vol of vol\n        self.rho = rho     # Correlation\n        self.r = r         # Risk-free rate\n    \n    def drift(self, t, X):\n        S, v = X[0], X[1]\n        return np.array([\n            self.r * S,                           # dS drift\n            self.kappa * (self.theta - v)         # dv drift\n        ])\n    \n    def diffusion(self, t, X):\n        S, v = X[0], X[1]\n        sqrt_v = np.sqrt(max(v, 0))  # Ensure non-negative\n        return np.array([\n            [sqrt_v * S, 0],                      # dS diffusion\n            [self.xi * sqrt_v * self.rho, \n             self.xi * sqrt_v * np.sqrt(1 - self.rho**2)]  # dv diffusion\n        ])\n\n# Monte Carlo option pricing with Heston\nheston = HestonModel(kappa=2.0, theta=0.04, xi=0.3, rho=-0.7, r=0.05)\nprice_paths, vol_paths = heston.simulate(\n    X0=[100, 0.04], T=1.0, n_steps=252, n_paths=10000\n)\n```\n\n### 8.3 Jump Process Implementation\n\n```python\nfrom quantlib_pro.jumps import MertonJumpDiffusion\n\nclass MertonModel:\n    def __init__(self, mu, sigma, lam, mu_J, sigma_J):\n        self.mu = mu        # Drift\n        self.sigma = sigma  # Diffusion volatility\n        self.lam = lam      # Jump intensity\n        self.mu_J = mu_J    # Jump mean\n        self.sigma_J = sigma_J  # Jump volatility\n    \n    def simulate_jumps(self, T, n_paths):\n        # Poisson arrivals\n        n_jumps = np.random.poisson(self.lam * T, n_paths)\n        \n        # Jump times and sizes\n        jump_times = []\n        jump_sizes = []\n        \n        for path in range(n_paths):\n            if n_jumps[path] > 0:\n                times = np.sort(np.random.uniform(0, T, n_jumps[path]))\n                sizes = np.random.normal(self.mu_J, self.sigma_J, n_jumps[path])\n                jump_times.append(times)\n                jump_sizes.append(sizes)\n            else:\n                jump_times.append(np.array([]))\n                jump_sizes.append(np.array([]))\n        \n        return jump_times, jump_sizes\n```\n\nThese stochastic process foundations enable sophisticated modeling of financial markets, providing the mathematical framework for derivatives pricing, risk management, and portfolio optimization in QuantLib Pro.
