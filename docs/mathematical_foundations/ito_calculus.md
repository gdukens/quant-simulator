# Itô Calculus and Stochastic Integration\n\nItô calculus provides the mathematical framework for analyzing stochastic differential equations and forms the foundation of modern quantitative finance. This chapter presents rigorous development of stochastic integration theory and its applications to financial modeling.\n\n## 1. Motivation and Heuristics\n\n### 1.1 The Need for Stochastic Calculus\n\nClassical calculus fails for stochastic processes due to:\n- **Infinite variation**: Brownian motion has infinite variation on any interval\n- **Non-differentiability**: Sample paths are nowhere differentiable\n- **Quadratic variation**: Non-zero quadratic variation requires new integration theory\n\n**Example**: For Brownian motion $W_t$, the \"derivative\" $dW_t/dt$ doesn't exist in the classical sense, yet we need to solve equations like:\n\n```{math}\n:label: motivation_sde\n\ndS_t = \\mu S_t dt + \\sigma S_t dW_t\n```\n\n### 1.2 Quadratic Variation Revisited\n\nFor a partition $\\Pi: 0 = t_0 < t_1 < \\cdots < t_n = T$ with mesh $|\\Pi| = \\max_i(t_{i+1} - t_i)$:\n\n**Definition 1.1** (Quadratic Variation): The quadratic variation of a process $(X_t)$ is:\n\n```{math}\n:label: quadratic_variation_def\n\n[X]_T = \\lim_{|\\Pi| \\to 0} \\sum_{i=0}^{n-1} (X_{t_{i+1}} - X_{t_i})^2\n```\n\nwhen the limit exists in probability.\n\n**Key Results**:\n- For Brownian motion: $[W]_t = t$\n- For finite variation processes: $[X]_t = 0$\n- **Itô isometry**: This will lead to $\\mathbb{E}[(\\int_0^T H_s dW_s)^2] = \\mathbb{E}[\\int_0^T H_s^2 ds]$\n\n## 2. Construction of the Stochastic Integral\n\n### 2.1 Simple Processes\n\n**Definition 2.1** (Simple Process): A process $H_t$ is simple if:\n\n```{math}\n:label: simple_process\n\nH_t = \\sum_{i=0}^{n-1} H_{t_i} \\mathbf{1}_{(t_i, t_{i+1}]}(t)\n```\n\nwhere $H_{t_i}$ is $\\mathcal{F}_{t_i}$-measurable and bounded.\n\n**Stochastic Integral for Simple Processes**:\n\n```{math}\n:label: simple_integral\n\n\\int_0^T H_s dW_s = \\sum_{i=0}^{n-1} H_{t_i} (W_{t_{i+1}} - W_{t_i})\n```\n\n### 2.2 Properties of Simple Integrals\n\n**Linearity**: \n```{math}\n:label: integral_linearity\n\n\\int_0^T (aH_s + bK_s) dW_s = a\\int_0^T H_s dW_s + b\\int_0^T K_s dW_s\n```\n\n**Itô Isometry**: \n```{math}\n:label: ito_isometry_simple\n\n\\mathbb{E}\\left[\\left(\\int_0^T H_s dW_s\\right)^2\\right] = \\mathbb{E}\\left[\\int_0^T H_s^2 ds\\right]\n```\n\n**Martingale Property**: If $\\int_0^T \\mathbb{E}[H_s^2] ds < \\infty$, then $\\int_0^t H_s dW_s$ is a martingale.\n\n### 2.3 Extension to General Integrands\n\n**Definition 2.2** ($L^2$ Space): Let $\\mathcal{L}^2[0,T]$ be the space of adapted processes with:\n\n```{math}\n:label: l2_norm\n\n\\|H\\|_{\\mathcal{L}^2}^2 = \\mathbb{E}\\left[\\int_0^T H_s^2 ds\\right] < \\infty\n```\n\n**Extension Theorem**: Simple processes are dense in $\\mathcal{L}^2[0,T]$, and the stochastic integral extends uniquely to $\\mathcal{L}^2[0,T]$ preserving:\n1. Linearity\n2. Itô isometry\n3. Martingale property\n\n### 2.4 Local Martingale Extension\n\nFor integrands $H$ with $\\int_0^T H_s^2 ds < \\infty$ a.s. (but possibly $\\mathbb{E}[\\int_0^T H_s^2 ds] = \\infty$):\n\n**Localization**: Define stopping times $\\tau_n = \\inf\\{t : \\int_0^t H_s^2 ds > n\\}$ and:\n\n```{math}\n:label: localized_integral\n\n\\int_0^{t \\wedge \\tau_n} H_s dW_s \\text{ is a martingale for each } n\n```\n\nThe stochastic integral is then a local martingale.\n\n## 3. Itô's Formula\n\n### 3.1 One-Dimensional Itô Formula\n\n**Theorem 3.1** (Itô's Formula): Let $X_t$ be an Itô process:\n\n```{math}\n:label: ito_process\n\nX_t = X_0 + \\int_0^t \\mu_s ds + \\int_0^t \\sigma_s dW_s\n```\n\nand $f \\in C^2(\\mathbb{R})$. Then:\n\n```{math}\n:label: ito_formula\n\nf(X_t) = f(X_0) + \\int_0^t f'(X_s) dX_s + \\frac{1}{2} \\int_0^t f''(X_s) d[X]_s\n```\n\nwhere $d[X]_s = \\sigma_s^2 ds$.\n\n**Differential Form**:\n\n```{math}\n:label: ito_differential\n\ndf(X_t) = f'(X_t) dX_t + \\frac{1}{2} f''(X_t) (dX_t)^2\n```\n\nwith the multiplication rule: $(dt)^2 = dt \\cdot dW_t = 0$ and $(dW_t)^2 = dt$.\n\n### 3.2 Multi-Dimensional Itô Formula\n\nFor $X_t = (X_t^1, \\ldots, X_t^d)$ with:\n\n```{math}\n:label: multid_ito_process\n\ndX_t^i = \\mu_t^i dt + \\sum_{j=1}^m \\sigma_t^{ij} dW_t^j\n```\n\nand $f \\in C^2(\\mathbb{R}^d)$:\n\n```{math}\n:label: multid_ito_formula\n\n\\begin{align}\ndf(X_t) &= \\sum_{i=1}^d \\frac{\\partial f}{\\partial x_i}(X_t) dX_t^i + \\frac{1}{2} \\sum_{i,j=1}^d \\frac{\\partial^2 f}{\\partial x_i \\partial x_j}(X_t) d[X^i, X^j]_t\nend{align}\n```\n\nwhere the cross-variation is:\n\n```{math}\n:label: cross_variation\n\nd[X^i, X^j]_t = \\sum_{k=1}^m \\sigma_t^{ik} \\sigma_t^{jk} dt\n```\n\n### 3.3 Applications of Itô's Formula\n\n#### 3.3.1 Geometric Brownian Motion\n\nFor $dS_t = \\mu S_t dt + \\sigma S_t dW_t$, apply Itô's formula to $f(x) = \\ln(x)$:\n\n```{math}\n:label: gbm_log_process\n\nd\\ln(S_t) = \\left(\\mu - \\frac{\\sigma^2}{2}\\right) dt + \\sigma dW_t\n```\n\n**Solution**:\n\n```{math}\n:label: gbm_solution\n\nS_t = S_0 \\exp\\left(\\left(\\mu - \\frac{\\sigma^2}{2}\\right)t + \\sigma W_t\\right)\n```\n\n#### 3.3.2 Ornstein-Uhlenbeck Process\n\nFor $dX_t = -\\lambda X_t dt + \\sigma dW_t$, use integrating factor $e^{\\lambda t}$:\n\n```{math}\n:label: ou_integrating_factor\n\nd(e^{\\lambda t} X_t) = \\sigma e^{\\lambda t} dW_t\n```\n\n**Solution**:\n\n```{math}\n:label: ou_solution_detailed\n\nX_t = X_0 e^{-\\lambda t} + \\sigma \\int_0^t e^{-\\lambda(t-s)} dW_s\n```\n\n## 4. Stochastic Differential Equations\n\n### 4.1 Existence and Uniqueness Theory\n\n**Standard Form**:\n\n```{math}\n:label: sde_standard\n\ndX_t = b(t, X_t) dt + \\sigma(t, X_t) dW_t, \\quad X_0 = x_0\n```\n\n**Theorem 4.1** (Existence and Uniqueness): If $b$ and $\\sigma$ satisfy:\n\n1. **Lipschitz Condition**: For some $K > 0$:\n   ```{math}\n   :label: lipschitz_condition\n   \n   |b(t,x) - b(t,y)| + |\\sigma(t,x) - \\sigma(t,y)| \\leq K|x - y|\n   ```\n\n2. **Growth Condition**: For some $K > 0$:\n   ```{math}\n   :label: growth_condition\n   \n   |b(t,x)|^2 + |\\sigma(t,x)|^2 \\leq K(1 + |x|^2)\n   ```\n\nThen there exists a unique strong solution on $[0,T]$.\n\n### 4.2 Solution Methods\n\n#### 4.2.1 Direct Integration\n\n**Linear SDEs**: For $dX_t = (a(t) X_t + b(t)) dt + (c(t) X_t + d(t)) dW_t$:\n\nUse integrating factor method or variation of constants.\n\n#### 4.2.2 Transformation Methods\n\n**Example**: For $dX_t = X_t dW_t$, the solution is:\n\n```{math}\n:label: exponential_bm\n\nX_t = X_0 \\exp\\left(W_t - \\frac{t}{2}\\right)\n```\n\nverified by applying Itô's formula to $\\ln(X_t)$.\n\n### 4.3 Feynman-Kac Formula\n\n**Theorem 4.2** (Feynman-Kac): Consider the PDE:\n\n```{math}\n:label: feynman_kac_pde\n\n\\frac{\\partial u}{\\partial t} + \\frac{1}{2} \\sigma^2(t,x) \\frac{\\partial^2 u}{\\partial x^2} + \\mu(t,x) \\frac{\\partial u}{\\partial x} - r(t,x) u + f(t,x) = 0\n```\n\nwith terminal condition $u(T,x) = g(x)$. The solution is:\n\n```{math}\n:label: feynman_kac_solution\n\nu(t,x) = \\mathbb{E}\\left[g(X_T) e^{-\\int_t^T r(s,X_s) ds} + \\int_t^T f(s,X_s) e^{-\\int_t^s r(\\tau,X_\\tau) d\\tau} ds \\Big| X_t = x\\right]\n```\n\nwhere $X_s$ solves $dX_s = \\mu(s,X_s) ds + \\sigma(s,X_s) dW_s$.\n\n**Application**: This connects PDEs to expectation calculations, fundamental for option pricing.\n\n## 5. Martingale Representation Theorem\n\n### 5.1 Brownian Filtration\n\n**Theorem 5.1** (Martingale Representation): Every martingale $(M_t)_{t \\in [0,T]}$ with respect to the Brownian filtration $\\{\\mathcal{F}_t^W\\}$ can be represented as:\n\n```{math}\n:label: martingale_representation\n\nM_t = M_0 + \\int_0^t H_s dW_s\n```\n\nfor some adapted process $H$ with $\\int_0^T H_s^2 ds < \\infty$ a.s.\n\n**Financial Interpretation**: Any contingent claim can be replicated by trading in the underlying asset and bond.\n\n### 5.2 Multi-Dimensional Case\n\nFor $d$-dimensional Brownian motion $(W_t^1, \\ldots, W_t^d)$:\n\n```{math}\n:label: multid_representation\n\nM_t = M_0 + \\sum_{i=1}^d \\int_0^t H_s^i dW_s^i\n```\n\n**Market Completeness**: If we have $d$ risky assets driven by $d$ Brownian motions, the market is complete.\n\n## 6. Change of Measure and Girsanov's Theorem\n\n### 6.1 Girsanov's Theorem (Detailed)\n\n**Setup**: Let $\\theta_t$ be an adapted process satisfying Novikov's condition:\n\n```{math}\n:label: novikov_detailed\n\n\\mathbb{E}\\left[\\exp\\left(\\frac{1}{2} \\int_0^T |\\theta_s|^2 ds\\right)\\right] < \\infty\n```\n\n**Radon-Nikodym Density**:\n\n```{math}\n:label: rn_density\n\nZ_t = \\exp\\left(-\\int_0^t \\theta_s \\cdot dW_s - \\frac{1}{2} \\int_0^t |\\theta_s|^2 ds\\right)\n```\n\n**Theorem 6.1**: Under $\\mathbb{Q}$ with $\\frac{d\\mathbb{Q}}{d\\mathbb{P}}\\big|_{\\mathcal{F}_T} = Z_T$:\n\n```{math}\n:label: girsanov_transform\n\n\\tilde{W}_t = W_t + \\int_0^t \\theta_s ds\n```\n\nis a $\\mathbb{Q}$-Brownian motion.\n\n### 6.2 Application: Risk-Neutral Valuation\n\n**Asset Price SDE**:\n\n```{math}\n:label: asset_sde\n\ndS_t = \\mu S_t dt + \\sigma S_t dW_t\n```\n\n**Market Price of Risk**: $\\theta_t = \\frac{\\mu - r}{\\sigma}$\n\n**Risk-Neutral SDE**:\n\n```{math}\n:label: risk_neutral_sde\n\ndS_t = r S_t dt + \\sigma S_t d\\tilde{W}_t\n```\n\nunder the risk-neutral measure $\\mathbb{Q}$.\n\n## 7. Numerical Methods\n\n### 7.1 Euler-Maruyama Scheme\n\n**Discretization**: For $dX_t = \\mu(X_t) dt + \\sigma(X_t) dW_t$:\n\n```{math}\n:label: euler_maruyama_detailed\n\nX_{n+1} = X_n + \\mu(X_n) \\Delta t + \\sigma(X_n) \\Delta W_n\n```\n\nwhere $\\Delta W_n \\sim \\mathcal{N}(0, \\Delta t)$.\n\n**Strong Convergence**: Order $\\frac{1}{2}$ in $\\Delta t$.\n\n**Weak Convergence**: Order $1$ in $\\Delta t$.\n\n### 7.2 Milstein Scheme\n\n**Higher-Order Correction**:\n\n```{math}\n:label: milstein_detailed\n\nX_{n+1} = X_n + \\mu(X_n) \\Delta t + \\sigma(X_n) \\Delta W_n + \\frac{1}{2} \\sigma(X_n) \\sigma'(X_n) [(\\Delta W_n)^2 - \\Delta t]\n```\n\n**Strong Convergence**: Order $1$ in $\\Delta t$.\n\n### 7.3 Predictor-Corrector Methods\n\n**Heun Scheme**:\n\n```{math}\n:label: heun_scheme\n\n\\begin{align}\n\\tilde{X}_{n+1} &= X_n + \\mu(X_n) \\Delta t + \\sigma(X_n) \\Delta W_n \\\\\nX_{n+1} &= X_n + \\frac{1}{2}[\\mu(X_n) + \\mu(\\tilde{X}_{n+1})] \\Delta t + \\sigma(X_n) \\Delta W_n\nend{align}\n```\n\n## 8.
