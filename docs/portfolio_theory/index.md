# Portfolio Theory and Asset Allocation\n\nPortfolio theory provides the mathematical framework for optimal asset allocation under risk and return trade-offs. This chapter presents rigorous development of modern portfolio theory, capital asset pricing models, and multi-factor frameworks.\n\n## 1. Modern Portfolio Theory (MPT)\n\n### 1.1 Mathematical Framework\n\n**Setup**: Consider $n$ risky assets with random returns $R_1, \\ldots, R_n$.\n\n**Portfolio Definition**: A portfolio is characterized by weights $w = (w_1, \\ldots, w_n)^T$ where:\n\n```{math}\n:label: portfolio_weights\n\n\\sum_{i=1}^n w_i = 1, \\quad w_i \\in \\mathbb{R}\n```\n\n**Portfolio Return**: \n\n```{math}\n:label: portfolio_return\n\nR_p = \\sum_{i=1}^n w_i R_i = w^T R\n```\n\nwhere $R = (R_1, \\ldots, R_n)^T$.\n\n**Expected Return**: \n\n```{math}\n:label: expected_portfolio_return\n\n\\mu_p = \\mathbb{E}[R_p] = w^T \\mu\n```\n\nwhere $\\mu = (\\mathbb{E}[R_1], \\ldots, \\mathbb{E}[R_n])^T$.\n\n**Portfolio Variance**: \n\n```{math}\n:label: portfolio_variance\n\n\\sigma_p^2 = \\text{Var}[R_p] = w^T \\Sigma w\n```\n\nwhere $\\Sigma = \\text{Cov}[R, R]$ is the covariance matrix.\n\n### 1.2 Mean-Variance Optimization\n\n#### 1.2.1 Global Minimum Variance Portfolio\n\n**Problem**: Minimize portfolio variance without return constraint:\n\n```{math}\n:label: gmv_problem\n\n\\min_{w} \\frac{1}{2} w^T \\Sigma w \\quad \\text{subject to} \\quad \\mathbf{1}^T w = 1\n```\n\nwhere $\\mathbf{1} = (1, 1, \\ldots, 1)^T$.\n\n**Lagrangian**: \n\n```{math}\n:label: gmv_lagrangian\n\n\\mathcal{L}(w, \\lambda) = \\frac{1}{2} w^T \\Sigma w - \\lambda (\\mathbf{1}^T w - 1)\n```\n\n**First-Order Conditions**: \n\n```{math}\n:label: gmv_foc\n\n\\begin{align}\n\\frac{\\partial \\mathcal{L}}{\\partial w} &= \\Sigma w - \\lambda \\mathbf{1} = 0 \\\\\n\\frac{\\partial \\mathcal{L}}{\\partial \\lambda} &= \\mathbf{1}^T w - 1 = 0\nend{align}\n```\n\n**Solution**: \n\n```{math}\n:label: gmv_solution\n\nw_{gmv} = \\frac{\\Sigma^{-1} \\mathbf{1}}{\\mathbf{1}^T \\Sigma^{-1} \\mathbf{1}}\n```\n\n**Minimum Variance**: \n\n```{math}\n:label: minimum_variance\n\n\\sigma_{gmv}^2 = \\frac{1}{\\mathbf{1}^T \\Sigma^{-1} \\mathbf{1}}\n```\n\n#### 1.2.2 Mean-Variance Efficient Portfolios\n\n**Problem**: For target expected return $\\mu_0$:\n\n```{math}\n:label: mv_efficient_problem\n\n\\min_{w} \\frac{1}{2} w^T \\Sigma w \\quad \\text{subject to} \\quad \\begin{cases}\n\\mathbf{1}^T w = 1 \\\\\n\\mu^T w = \\mu_0\nend{cases}\n```\n\n**Lagrangian**: \n\n```{math}\n:label: mv_lagrangian\n\n\\mathcal{L}(w, \\lambda_1, \\lambda_2) = \\frac{1}{2} w^T \\Sigma w - \\lambda_1 (\\mathbf{1}^T w - 1) - \\lambda_2 (\\mu^T w - \\mu_0)\n```\n\n**Solution**: \n\n```{math}\n:label: mv_efficient_solution\n\nw^*(\\mu_0) = g + h \\mu_0\n```\n\nwhere:\n\n```{math}\n:label: gh_vectors\n\n\\begin{align}\ng &= \\frac{B \\Sigma^{-1} \\mathbf{1} - A \\Sigma^{-1} \\mu}{\\Delta} \\\\\nh &= \\frac{C \\Sigma^{-1} \\mu - A \\Sigma^{-1} \\mathbf{1}}{\\Delta}\nend{align}\n```\n\n**Constants**: \n\n```{math}\n:label: abc_constants\n\n\\begin{align}\nA &= \\mathbf{1}^T \\Sigma^{-1} \\mu \\\\\nB &= \\mu^T \\Sigma^{-1} \\mu \\\\\nC &= \\mathbf{1}^T \\Sigma^{-1} \\mathbf{1} \\\\\n\\Delta &= BC - A^2\nend{align}\n```\n\n### 1.3 The Efficient Frontier\n\n**Parametric Representation**: The efficient frontier in $$(\\sigma_p, \\mu_p)$$ space is:\n\n```{math}\n:label: efficient_frontier\n\n\\sigma_p^2(\\mu_p) = \\frac{1}{C} + \\frac{(\\mu_p - A/C)^2}{\\Delta/C}\n```\n\nThis is a hyperbola opening rightward with vertex at:\n\n```{math}\n:label: frontier_vertex\n\n\\left(\\sqrt{\\frac{1}{C}}, \\frac{A}{C}\\right)\n```\n\n**Two-Fund Theorem**: Every efficient portfolio is a combination of any two distinct efficient portfolios.\n\n**Proof**: Let $w_1$ and $w_2$ be two efficient portfolios. Any combination:\n\n```{math}\n:label: two_fund_combination\n\nw = \\alpha w_1 + (1-\\alpha) w_2\n```\n\nis also efficient for appropriate choice of $\\alpha$.\n\n### 1.4 Risk-Free Asset Extension\n\n**Setup**: Include risk-free asset with return $r_f$ and zero variance.\n\n**Portfolio Return**: \n\n```{math}\n:label: portfolio_return_rf\n\nR_p = w_0 r_f + \\sum_{i=1}^n w_i R_i\n```\n\nwith constraint $w_0 + \\sum_{i=1}^n w_i = 1$.\n\n**Optimization Problem**: \n\n```{math}\n:label: rf_optimization\n\n\\max_{w} \\frac{w^T (\\mu - r_f \\mathbf{1})}{\\sqrt{w^T \\Sigma w}}\n```\n\n**Tangency Portfolio**: The optimal risky portfolio is:\n\n```{math}\n:label: tangency_portfolio\n\nw_t = \\frac{\\Sigma^{-1}(\\mu - r_f \\mathbf{1})}{\\mathbf{1}^T \\Sigma^{-1}(\\mu - r_f \\mathbf{1})}\n```\n\n**Sharpe Ratio**: The maximum Sharpe ratio is:\n\n```{math}\n:label: max_sharpe_ratio\n\nSR_{max} = \\sqrt{(\\mu - r_f \\mathbf{1})^T \\Sigma^{-1} (\\mu - r_f \\mathbf{1})}\n```\n\n### 1.5 Capital Allocation Line (CAL)\n\n**Equation**: Combinations of risk-free asset and tangency portfolio:\n\n```{math}\n:label: cal_equation\n\n\\mu_p = r_f + SR_{max} \\cdot \\sigma_p\n```\n\n**Investor's Problem**: Choose allocation $\\alpha$ to risky assets:\n\n```{math}\n:label: investor_allocation\n\n\\max_\\alpha \\mathbb{E}[U(\\alpha R_t + (1-\\alpha) r_f)]\n```\n\nwhere $U$ is the investor's utility function.\n\n## 2. Capital Asset Pricing Model (CAPM)\n\n### 2.1 Assumptions and Setup\n\n**CAPM Assumptions**:\n1. Investors are mean-variance optimizers\n2. Homogeneous expectations\n3. Single-period model\n4. Risk-free borrowing and lending\n5. No transaction costs or taxes\n6. Price-taking behavior\n\n**Market Portfolio**: In equilibrium, all investors hold the same risky portfolio - the market portfolio $M$.\n\n### 2.2 Security Market Line (SML)\n\n**Beta Definition**: For asset $i$:\n\n```{math}\n:label: beta_definition\n\n\\beta_i = \\frac{\\text{Cov}[R_i, R_M]}{\\text{Var}[R_M]} = \\frac{\\sigma_{i,M}}{\\sigma_M^2}\n```\n\n**CAPM Equation**: \n\n```{math}\n:label: capm_equation\n\n\\mathbb{E}[R_i] = r_f + \\beta_i (\\mathbb{E}[R_M] - r_f)\n```\n\n**Risk Premium Decomposition**: \n\n```{math}\n:label: risk_premium_decomposition\n\n\\mathbb{E}[R_i] - r_f = \\beta_i (\\mathbb{E}[R_M] - r_f)\n```\n\n### 2.3 CAPM Derivation\n\n**Equilibrium Condition**: In market equilibrium, supply equals demand for all assets. The market portfolio $M$ must be mean-variance efficient.\n\n**Mathematical Derivation**: For any asset $i$ and the market portfolio $M$:\n\n```{math}\n:label: capm_derivation\n\n\\frac{\\mathbb{E}[R_i] - r_f}{\\text{Cov}[R_i, R_M]} = \\frac{\\mathbb{E}[R_M] - r_f}{\\text{Var}[R_M]}\n```\n\nRearranging yields the CAPM equation.\n\n### 2.4 Portfolio Beta\n\nFor a portfolio with weights $w$:\n\n```{math}\n:label: portfolio_beta\n\n\\beta_p = \\sum_{i=1}^n w_i \\beta_i = w^T \\beta\n```\n\nwhere $\\beta = (\\beta_1, \\ldots, \\beta_n)^T$.\n\n### 2.5 Security Characteristic Line (SCL)\n\nRegression of asset returns on market returns:\n\n```{math}\n:label: scl_regression\n\nR_i = \\alpha_i + \\beta_i R_M + \\epsilon_i\n```\n\nwhere:\n- $\\alpha_i$: Jensen's alpha (abnormal return)\n- $\\epsilon_i$: Idiosyncratic risk with $\\mathbb{E}[\\epsilon_i] = 0$ and $\\text{Cov}[\\epsilon_i, R_M] = 0$\n\n**Under CAPM**: $\\alpha_i = 0$ for all assets.\n\n### 2.6 Decomposition of Risk\n\n**Total Risk**: \n\n```{math}\n:label: total_risk_decomposition\n\n\\text{Var}[R_i] = \\beta_i^2 \\text{Var}[R_M] + \\text{Var}[\\epsilon_i]\n```\n\n**Systematic Risk**: $\\beta_i^2 \\sigma_M^2$ (cannot be diversified away)\n\n**Idiosyncratic Risk**: $\\text{Var}[\\epsilon_i]$ (can be diversified away)\n\n## 3. Arbitrage Pricing Theory (APT)\n\n### 3.1 Factor Model Framework\n\n**K-Factor Model**: \n\n```{math}\n:label: k_factor_model\n\nR_i = \\mathbb{E}[R_i] + \\sum_{j=1}^K \\beta_{ij} F_j + \\epsilon_i\n```\n\nwhere:\n- $F_j$: Factor $j$ with $\\mathbb{E}[F_j] = 0$\n- $\\beta_{ij}$: Loading of asset $i$ on factor $j$\n- $\\epsilon_i$: Idiosyncratic error with $\\mathbb{E}[\\epsilon_i] = 0$\n\n**Factor Structure Assumptions**:\n1. $\\text{Cov}[F_j, \\epsilon_i] = 0$ for all $i, j$\n2. $\\text{Cov}[\\epsilon_i, \\epsilon_j] = 0$ for $i \\neq j$ (or negligible for large portfolios)\n\n### 3.2 APT Derivation\n\n**No-Arbitrage Condition**: Consider a well-diversified portfolio that eliminates idiosyncratic risk.\n\n**Approximate Factor Model**: For large portfolios:\n\n```{math}\n:label: portfolio_factor_model\n\nR_p \\approx \\mathbb{E}[R_p] + \\sum_{j=1}^K \\beta_{pj} F_j\n```\n\nwhere $\\beta_{pj} = \\sum_{i=1}^n w_i \\beta_{ij}$.\n\n**Arbitrage Argument**: If two portfolios have the same factor loadings but different expected returns, an arbitrage opportunity exists.\n\n**APT Equation**: \n\n```{math}\n:label: apt_equation\n\n\\mathbb{E}[R_i] = r_f + \\sum_{j=1}^K \\beta_{ij} \\lambda_j\n```\n\nwhere $\\lambda_j$ is the risk premium for factor $j$.\n\n### 3.3 Factor Risk Premiums\n\n**Determination**: Risk premiums $\\lambda_j$ are determined by:\n\n```{math}\n:label: factor_risk_premiums\n\n\\lambda_j = \\mathbb{E}[R_{F_j}] - r_f\n```\n\nwhere $R_{F_j}$ is the return on a portfolio with $\\beta_{Fj,j} = 1$ and $\\beta_{Fj,k} = 0$ for $k \\neq j$.\n\n### 3.4 Multi-Factor Models\n\n#### 3.4.1 Fama-French Three-Factor Model\n\n```{math}\n:label: fama_french_3factor\n\nR_i - r_f = \\alpha_i + \\beta_i (R_M - r_f) + s_i \\text{SMB} + h_i \\text{HML} + \\epsilon_i\n```\n\nwhere:\n- SMB: Small Minus Big (size factor)\n- HML: High Minus Low (value factor)\n\n#### 3.4.2 Fama-French Five-Factor Model\n\n```{math}\n:label: fama_french_5factor\n\nR_i - r_f = \\alpha_i + \\beta_i (R_M - r_f) + s_i \\text{SMB} + h_i \\text{HML} + r_i \\text{RMW} + c_i \\text{CMA} + \\epsilon_i\n```\n\nwhere:\n- RMW: Robust Minus Weak (profitability factor)\n- CMA: Conservative Minus Aggressive (investment factor)\n\n## 4. Black-Litterman Model\n\n### 4.1 Motivation\n\nTraditional mean-variance optimization suffers from:\n- **Estimation error**: Sample means are noisy\n- **Extreme portfolios**: Small changes in inputs lead to large portfolio changes\n- **Unintuitive results**: Optimizer may suggest counterintuitive positions\n\n### 4.2 Bayesian Framework\n\n**Prior Beliefs**: Start with CAPM equilibrium expected returns:\n\n```{math}\n:label: bl_prior_returns\n\n\\Pi = \\lambda \\Sigma w_{market}\n```\n\nwhere $\\lambda$ is the risk aversion parameter.\n\n**Prior Distribution**: \n\n```{math}\n:label: bl_prior_distribution\n\n\\mu \\sim \\mathcal{N}(\\Pi, \\tau \\Sigma)\n```\n\nwhere $\\tau$ controls confidence in the prior.\n\n### 4.3 Incorporating Views\n\n**Views Matrix**: Express $k$ views as:\n\n```{math}\n:label: bl_views_matrix\n\nP \\mu = Q + \\epsilon\n```\n\nwhere:\n- $P$: $k \\times n$ matrix identifying assets in each view\n- $Q$: $k \\times 1$ vector of view returns\n- $\\epsilon \\sim \\mathcal{N}(0, \\Omega)$: view errors\n\n### 4.4 Bayesian Update\n\n**Posterior Distribution**: \n\n```{math}\n:label: bl_posterior\n\n\\mu | \\text{views} \\sim \\mathcal{N}(\\mu_{BL}, \\Sigma_{BL})\n```\n\n**Black-Litterman Expected Returns**: \n\n```{math}\n:label: bl_expected_returns\n\n\\mu_{BL} = [(\\tau \\Sigma)^{-1} + P^T \\Omega^{-1} P]^{-1} [(\\tau \\Sigma)^{-1} \\Pi + P^T \\Omega^{-1} Q]\n```\n\n**Black-Litterman Covariance**: \n\n```{math}\n:label: bl_covariance\n\n\\Sigma_{BL} = [(\\tau \\Sigma)^{-1} + P^T \\Omega^{-1} P]^{-1}\n```\n\n### 4.5 Optimal Portfolio\n\nUsing Black-Litterman inputs in mean-variance optimization:\n\n```{math}\n:label: bl_optimal_portfolio\n\nw_{BL} = \\frac{1}{\\lambda} \\Sigma^{-1} \\mu_{BL}\n```\n\n## 5. Risk-Based Portfolio Construction\n\n### 5.1 Risk Parity\n\n**Equal Risk Contribution**: Allocate capital so each asset contributes equally to portfolio risk.\n\n**Risk Contribution**: For asset $i$:\n\n```{math}\n:label: risk_contribution\n\nRC_i = w_i \\frac{\\partial \\sigma_p}{\\partial w_i} = w_i \\frac{(\\Sigma w)_i}{\\sigma_p}\n```\n\n**Risk Parity Condition**: \n\n```{math}\n:label: risk_parity_condition\n\nRC_i = \\frac{\\sigma_p}{n} \\quad \\forall i\n```\n\n**Optimization Problem**: \n\n```{math}\n:label: risk_parity_optimization\n\n\\min_w \\sum_{i=1}^n \\left(w_i \\frac{(\\Sigma w)_i}{w^T \\Sigma w} - \\frac{1}{n}\\right)^2 \\quad \\text{s.t.} \\quad \\mathbf{1}^T w = 1, \\, w \\geq 0\n```\n\n### 5.2 Maximum Diversification\n\n**Diversification Ratio**: \n\n```{math}\n:label: diversification_ratio\n\nDR = \\frac{w^T \\sigma}{\\sqrt{w^T \\Sigma w}}\n```\n\nwhere $\\sigma = (\\sigma_1, \\ldots, \\sigma_n)^T$ are individual asset volatilities.\n\n**Optimization**: \n\n```{math}\n:label: max_diversification\n\n\\max_w \\frac{w^T \\sigma}{\\sqrt{w^T \\Sigma w}} \\quad \\text{s.t.} \\quad \\mathbf{1}^T w = 1, \\, w \\geq 0\n```\n\n### 5.3 Minimum Variance\n\nSimplest risk-based approach - ignore expected returns entirely:\n\n```{math}\n:label: min_variance_portfolio\n\n\\min_w w^T \\Sigma w \\quad \\text{s.t.} \\quad \\mathbf{1}^T w = 1\n```\n\n## 6.
